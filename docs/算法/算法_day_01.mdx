<br/>


<h3>数据</h3>
<p>
是计算机中可操作的对象,是能被计算机识别,并输入给计算机的符号集合
(整型,字符型,MP3文件等等)
</p>
<br/>

<h3>数据元素</h3>
<p>
是组成数据的有一定意义的基本单位,在计算机中通常作为整体处理,也叫做<code>记录</code>
</p>

<br/>
<br/>
<h3>数据项</h3>
<p>
一个数据元素可以由若干个数据项组成,如人的身体可以由耳朵和鼻子等组成
<em>数据项是数据不可分割的最小单位</em>
</p>

<br/>
<br/>
<h3>数据对象</h3>
<p>
是性质相同的数据元素的集合,是数据的子集,比如人的姓名,生日
</p>

<br/>
<br/>
<h3>数据结构</h3>
<p>
是相互之间存在一种或多种关系的数据元素的集合
</p>
